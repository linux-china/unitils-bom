Unitils BOM
============================
unitils bom

### Usage

in your pom.xml: 

       <dependency>
            <groupId>org.unitils</groupId>
            <artifactId>unitils-bom</artifactId>
            <version>3.4.2</version>
            <type>pom</type>
            <scope>test</scope>
       </dependency>
